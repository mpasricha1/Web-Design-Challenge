Summary: Latitude vs. X

Welcome to the weather data analysis page. The purpose of the project was to analysis weather data as you get closer to the Equator. To build the dataset, data was pulled in from the OpenWeatherMap API. After the data was assembled from the API, there were several other libraries that were used in order to plot the data. A DateFrame of the data was created using the Python library Pandas. Once in the DataFrame, the data was able to be cleaned and manipulated to plot several different observations.

To plot the data, the Python library Matplotlib was used. This allowed us to create several different plots. The main focus of the ask were several weather factors and how they related to latitude. The first plot was a comparison of latitude vs temperature. The next plot analysed latitude vs cloudiness. The third was latitude vs humidity, and finally latitude vs wind speed. After the data was analysed for each individule plot, Matplotlib took over in order to create the plots. Each plot is a scatter plot for its relative data. In the side bar you can view a preview of each plot. You can click on any image to take you to the a larger image and summary of each plot.
